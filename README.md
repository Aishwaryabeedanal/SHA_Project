# Solution Health Analyzer (SHA)

A Visual Studio extension that analyzes solution health in real time.

## Features
- Roslyn-based error and warning detection
- TODO comment scanning
- Health score calculation
- NuGet package inspection using IVsPackageInstallerServices
- Live NuGet API for latest version checking
- Pre-release and outdated package detection
- AI-powered recommendations using Gemini API
- MCP server for natural language commands (scan, build, clean)

## Tech Stack
- C# / .NET Framework
- Visual Studio SDK
- Roslyn API
- NuGet v3 API
- Gemini AI API
- WPF / XAML
- HttpListener (MCP Server)
